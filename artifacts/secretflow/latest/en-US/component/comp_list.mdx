Component version: 0.0.1

Statistics evaluation for a bi-classification model on a dataset.

1. summary\_report: SummaryReport
2. group\_reports: List\[GroupReport]
3. eq\_frequent\_bin\_report: List\[EqBinReport]
4. eq\_range\_bin\_report: List\[EqBinReport]
5. head\_report: List\[PrReport]
   reports for fpr = 0.001, 0.005, 0.01, 0.05, 0.1, 0.2
