      <Outline.Content>
        Returns the config of the <cite>Model</cite>.

        Config is a Python dictionary (serializable) containing the
        configuration of an object, which in this case is a <cite>Model</cite>. This allows
        the <cite>Model</cite> to be be reinstantiated later (without its trained weights)
        from this configuration.

        Note that <cite>get\_config()</cite> does not guarantee to return a fresh copy of
        dict every time it is called. The callers should make a copy of the
        returned dict if they want to modify it.

        Developers of subclassed <cite>Model</cite> are advised to override this method,
        and continue to update the dict from <cite>super(MyModel, self).get\_config()</cite>
        to provide the proper configuration of this <cite>Model</cite>. The default config
        is an empty dict. Optionally, raise <cite>NotImplementedError</cite> to allow Keras
        to attempt a default serialization.
