# 水平联邦

:target{#what-is-horizontal-federated-learning}

## 水平联邦的概念

水平联邦模式适用于参与方业务相同但是客户群体不同的场景。该场景下，可以将来自各方的样本联合训练得到一个相比单方训练表现更好的模型。例如在医疗场景中，每个医院都有自己的患者群体，而不同地区的医院患者大部分是不重合的，但是患者的医疗数据（比如影像、验血结果等），其数据形式可能是相同的。

![](../../../_assets/federate_learning.png)

水平联邦训练的典型流程：

1. 每个参与方从服务端下载最新的模型。
2. 每个参与方使用自己的数据训练模型，并且将更新量加密上传到服务端。
3. 服务端对加密的更新量进行安全聚合，并更新模型。
4. 服务端把更新后的模型发给每个参与方。
5. 每个参与方更新本地模型，并重复进行下一次迭代。
