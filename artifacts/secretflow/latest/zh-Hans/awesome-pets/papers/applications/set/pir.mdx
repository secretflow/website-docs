# Private Information Retrieval (PIR)

![](https://badgen.net/badge/\:update-to/\:Mar-2023/red) ![](https://badgen.net/badge/\:papers/:33/blue)

Private Information Retrival is a protocol that allows a client to retrieve an element of a database without the owner of that database being able to determine which element was selected. Note that the initial motivation of PIR is to let users to privately download a website from the public internet, hence the database in PIR is assumed to be <strong>public</strong> and <strong>huge</strong>, this should be the explanation why early researchers are trying to find a asymptotically better (sublinear) solution in stead of a concretely better one.
