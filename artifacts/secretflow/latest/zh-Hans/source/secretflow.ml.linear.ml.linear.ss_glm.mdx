              - <strong>batch\_size</strong> – int, default=1024
                how many samples use in one calculation.
              - <strong>iter\_start\_irls</strong> – int, default=0
                run a few rounds of irls training as the initialization of w, 0 disable.
              - <strong>eps</strong> – float, default=1e-4

              - <strong>y</strong> – \{FedNdarray, VDataFrame} of shape (n\_samples,)
                Target vector relative to X.
              - <strong>offset</strong> – \{FedNdarray, VDataFrame} of shape (n\_samples,)
                Specify a column to use as the offset, Offsets are per-row “bias values” that are used during model training.

              - <strong>link</strong> – str

              - <strong>batch\_size</strong> – int, default=1024
                how many samples use in one calculation.
              - <strong>iter\_start\_irls</strong> – int, default=0
                run a few rounds of irls training as the initialization of w, 0 disable.
              - <strong>eps</strong> – float, default=1e-4
