        Horizontal federated training interface

        <FieldList>
          <dl>
            <dt className="field-name">
              参数
            </dt>

            <dd className="field-body">
              - <strong>x</strong> – feature, FedNdArray, HDataFrame or Dict \{PYU: model\_path}
              - <strong>y</strong> – label, FedNdArray, HDataFrame or str(column name of label)
              - <strong>batch\_size</strong> – Number of samples per gradient update, int or Dict, recommend 64 or more for safety
              - <strong>batch\_sampling\_rate</strong> – Ratio of sample per batch, float
              - <strong>epochs</strong> – Number of epochs to train the model
              - <strong>verbose</strong> – 0, 1. Verbosity mode
              - <strong>callbacks</strong> – List of <cite>keras.callbacks.Callback</cite> instances.
              - <strong>validation\_data</strong> – Data on which to evaluate
              - <strong>shuffle</strong> – whether to shuffle the training data
              - <strong>class\_weight</strong> – Dict mapping class indices (integers) to a weight (float)
              - <strong>sample\_weight</strong> – weights for the training samples
              - <strong>validation\_freq</strong> – specifies how many training epochs to run before a new validation run is performed
              - <strong>aggregate\_freq</strong> – Number of steps of aggregation
              - <strong>label\_decoder</strong> – Only used for CSV reading, for label preprocess
              - <strong>max\_batch\_size</strong> – Max limit of batch size
              - <strong>prefetch\_buffer\_size</strong> – An int specifying the number of feature batches to prefetch for performance improvement. Only for csv reader
              - <strong>sampler\_method</strong> – The name of sampler method
              - <strong>random\_seed</strong> – Prg seed for shuffling
              - <strong>dp\_spent\_step\_freq</strong> – specifies how many training steps to check the budget of dp
              - <strong>audit\_log\_dir</strong> – path of audit log dir, checkpoint will be save if audit\_log\_dir is not None
              - <strong>dataset\_builder</strong> – Callable function about hot to build the dataset. must return (dataset, steps\_per\_epoch)
              - <strong>wait\_steps</strong> – A step size to indicate how many concurrent tasks should be waited, which could prevent the stuck of ray when more tasks join (default 100).
            </dd>

            <dt className="field-name">
              返回
            </dt>

            <dd className="field-body">
              A history object. It’s history.global\_history attribute is a
              aggregated record of training loss values and metrics, while
              history.local\_history attribute is a record of training loss
              values and metrics of each party.
            </dd>
          </dl>
        </FieldList>
      </Outline.Content>
