            Specifies the minimum frequency below which a category will be
            considered infrequent.
            - If <cite>int</cite>, categories with a smaller cardinality will be considered
            infrequent.
            - If <cite>float</cite>, categories with a smaller cardinality than
            <cite>min\_frequency \* n\_samples</cite>  will be considered infrequent.
          - <strong>max\_categories</strong> – int, default=None
            Specifies an upper limit to the number of output features for each input
            feature when considering infrequent categories. If there are infrequent
            categories, <cite>max\_categories</cite> includes the category representing the
            infrequent categories along with the frequent categories. If <cite>None</cite>,
            there is no limit to the number of output features.

  <Outline.Content>
    基类：`_PreprocessBase`

    Standardize features by removing the mean and scaling to unit variance.

    StandardScaler is similar to [`sklearn.preprocessing.StandardScaler`](https://scikit-learn.org/stable/modules/generated/sklearn.preprocessing.StandardScaler.html#sklearn.preprocessing.StandardScaler "（在 scikit-learn v1.3）").
    The main differences are
    a) takes HDataFrame/VDataFrame/MixDataFrame as input/output.
    b) does not support sparse matrix.

    The standard score of a sample <cite>x</cite> is calculated as:

    > z = (x - u) / s

    where <cite>u</cite> is the mean of the training samples or zero if <cite>with\_mean=False</cite>,
    and <cite>s</cite> is the standard deviation of the training samples or one if
    <cite>with\_std=False</cite>.

        <dd className="field-body">
          - <strong>min\_frequency</strong> –&#x20;

            int or float, default=None
            Specifies the minimum frequency below which a category will be
            considered infrequent.
            - If <cite>int</cite>, categories with a smaller cardinality will be considered
            infrequent.
            - If <cite>float</cite>, categories with a smaller cardinality than
            <cite>min\_frequency \* n\_samples</cite>  will be considered infrequent.
          - <strong>max\_categories</strong> – int, default=None
            Specifies an upper limit to the number of output features for each input
            feature when considering infrequent categories. If there are infrequent
            categories, <cite>max\_categories</cite> includes the category representing the
            infrequent categories along with the frequent categories. If <cite>None</cite>,
            there is no limit to the number of output features.

<Outline domain="py" objectType="class" target={"secretflow.preprocessing.StandardScaler"} module={"secretflow.preprocessing"} fullname={"StandardScaler"} parameters={["with_mean", "with_std"]} description={"\u57fa\u7c7b\uff1a_PreprocessBase"}>
  <Outline.Signature fullname="StandardScaler" id="secretflow.preprocessing.StandardScaler">
    <Outline.Keyword>class<span>{" "}</span></Outline.Keyword><Outline.Prefix>secretflow\.preprocessing.</Outline.Prefix><Outline.Name>StandardScaler</Outline.Name><Outline.ParameterList><span>(</span><Outline.Parameter><span className="name">with\_mean</span><span>=</span><span>True</span></Outline.Parameter>, <Outline.Parameter><span className="name">with\_std</span><span>=</span><span>True</span></Outline.Parameter><span>)</span></Outline.ParameterList>
  </Outline.Signature>

  <Outline.Content>
    基类：`_PreprocessBase`

    Standardize features by removing the mean and scaling to unit variance.

    StandardScaler is similar to [`sklearn.preprocessing.StandardScaler`](https://scikit-learn.org/stable/modules/generated/sklearn.preprocessing.StandardScaler.html#sklearn.preprocessing.StandardScaler "（在 scikit-learn v1.3）").
    The main differences are
    a) takes HDataFrame/VDataFrame/MixDataFrame as input/output.
    b) does not support sparse matrix.

    The standard score of a sample <cite>x</cite> is calculated as:

    > z = (x - u) / s

    where <cite>u</cite> is the mean of the training samples or zero if <cite>with\_mean=False</cite>,
    and <cite>s</cite> is the standard deviation of the training samples or one if
    <cite>with\_std=False</cite>.
