# <Target id="id1" />快速开始

欢迎来到隐语SecretFlow！

请根据 [部署](deployment.mdx#deployment) 和 [安装](installation.mdx#installation) 准备好SecretFlow。

然后，请参考 [教程](../tutorial/index.mdx#tutorial) 尝试SecretFlow。

接下来，请查看 [用户指南](../user_guide/index.mdx#user-guide) ，这里介绍了第一方的算法，工具和模块便于你开发。

如果你想要在你的隐私保护系统中通过统一的接口集成隐语算法，请查阅 [组件](../component/index.mdx#component) 。

最后，在你编写SecretFlow程序时请参考 [开发者](../developer/index.mdx#developer) 和 [API文档](../api/index.mdx#api)。
