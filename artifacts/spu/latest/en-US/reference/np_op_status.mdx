# JAX NumPy Operators Status

:target{#overview}

# Overview

SPU recommends users to use JAX as frontend language to write logics. We found most of users would utilize <strong>jax.numpy</strong> modules in their programs. We have conducted tests with some <em>selected</em> Operators for reference.

Just keep in mind, if you couldn’t find a <strong>jax.numpy</strong> operator in this list, it doesn’t mean it’s not supported. We just haven’t test it, e.g. jax.numpy.sort. And we don’t test other <strong>JAX</strong> modules at this moment.

Please contact us if

- You need to confirm the status of another <strong>jax.numpy</strong> operator not listed here.
- You find a <strong>jax.numpy</strong> is not working even it is marked as <strong>PASS</strong>. e.g. The precision is bad.
