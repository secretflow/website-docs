:target{#id3}

## 安装

:target{#pip}

### 通过 Pip 安装

HEU 安装包已经发布到了 PyPi，您可以通过 pip 工具安装 HEU

```bash
pip install sf-heu
```

:target{#id4}

### 从源码安装

如果您需要使用最新测试版本，可以从 [源码](https://github.com/secretflow/heu) 编译安装。

<DefinitionList>
  <dl>
    <dt>
      <DefinitionList.Term><span>依赖工具：</span></DefinitionList.Term>
    </dt>

    <dd>
      - [Bazel](https://docs.bazel.build/versions/main/install.html)
      - GCC >= 11
    </dd>
  </dl>
</DefinitionList>

编译和安装：

```bash
git clone --recursive git@github.com:secretflow/heu.git
cd heu
sh build_wheel_entrypoint.sh
```

:target{#id6}

## 测试运行

安装完成后，建议执行以下命令检查 HEU 是否工作正常

```bash
python -c "from heu import phe"
```

上述命令，如果没有任何输出，则说明安装成功；反之如果报错，则说明安装未成功，请参考 [FAQ](faq.mdx)
