# SCQL Documentation

Secure Collaborative Query Language (SCQL) is a system that translates SQL statements into a hybrid MPC-plaintext execution graph and executes them on a federation of database systems. The MPC framework is powered by [SPU](https://github.com/secretflow/spu).
