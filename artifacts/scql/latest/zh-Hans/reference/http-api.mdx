# SCQL API

SCQL 支持两种类型的 API：同步和异步。同步接口适用于执行快速查询，例如 DDL、DML 和简单的 DQL。而异步接口则推荐用于耗时较多的 DQL。

:target{#asynchronous}

## <Target id="async-api" />异步接口

:target{#public-submit-query}

#### 示例

如果需要执行 <cite>show databases</cite>，请求和响应的消息结构应该如下所示：

- 请求

```javascript
{
    "user": {
        "user": {
            "account_system_type": "NATIVE_USER",
            "native_user": { "name": "someuser", "password": "somepassword" }
        }
    },
    "query": "show databases;",
    "biz_request_id": "1234"
}
```

- 响应

```javascript
{
    "status": {
        "code": 0,
        "message": "",
        "details": []
    },
    "scdb_session_id": "some_session_id"
}
```

#### 示例

如果需要获取 <cite>show databases</cite> 的结果，可以发送如下的请求。

```javascript
{
    "user": {
        "user": {
            "account_system_type": "NATIVE_USER",
            "native_user": { "name": "some_user", "password": "some_password" }
        }
    },
    "scdb_session_id": "some_session_id"
}
```

如果成功，将收到带有状态码 0 的响应，如下所示：

```javascript
{
    "status": {
        "code": 0,
        "message": "",
        "details": []
    },
    "out_columns": [
        {
            "name": "Database",
            "shape": {
                "dim": [
                    {
                        "dim_value": "1"
                    },
                    {
                        "dim_value": "1"
                    }
                ]
            },
            "elem_type": "STRING",
            "option": "VALUE",
            "annotation": null,
            "int32_data":[],
            "int64_data":[],
            "float_data":[],
            "double_data":[],
            "bool_data":[],
            "string_data": ["scdb"]
        }
    ],
    "scdb_session_id": "some_session_id",
    "affected_rows": "0"
}
```

如果结果尚未准备好，响应将会是：

```javascript
{
    "status": {
        "code": 104,
        "message": "result not ready, please retry later",
        "details": []
    },
    "out_columns": [],
    "scdb_session_id": "some_session_id",
    "affected_rows": "0"
}
```
