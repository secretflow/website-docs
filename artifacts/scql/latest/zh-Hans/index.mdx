# SCQL 文档

Secure Collaborative Query Language (SCQL) 是一种能够将 SQL 语句转换为明密文混合执行图，并在联合数据库系统上执行的系统。该系统的 MPC 框架由 [SPU](https://github.com/secretflow/spu) 提供支持。
