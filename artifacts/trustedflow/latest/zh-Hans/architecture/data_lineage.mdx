# 数据血缘

数据血缘是指在TrustedFlow流转的数据可以追溯其原始数据来源和计算链路。数据血缘可以作为计算结果正确性校验、审计等的重要依据。

如图所示，TrustedFlow会在数据中记录产生该数据的所有祖先以及历经的计算代码，并且提供完整性保护。
